# 机器生成文本检测系统 使用说明

## 1. 系统简介

本系统是一个基于决策树算法的中文机器生成文本检测工具，能够判断输入的文本是由人类书写还是由AI（如ChatGPT等大语言模型）生成。系统以Web应用形式提供服务，用户通过浏览器即可使用，无需安装任何客户端软件。

**系统地址：** `http://127.0.0.1:5000`

**主要功能：**
- 文本人机来源检测（人类/AI生成）
- 检测概率与置信度展示
- 可解释性特征分析（展示影响判断的关键特征）

---

## 2. 启动系统

### 2.1 环境准备

首次使用前，需安装Python依赖：

```bash
# 进入项目目录
cd .....

# 创建虚拟环境（仅首次）
python -m venv .venv

# 激活虚拟环境
.venv\Scripts\activate

# 安装依赖（仅首次）
pip install -r requirements.txt
```

### 2.2 模型训练

首次使用或需要重新训练模型时，执行以下命令：

```bash
# 构建训练数据（从HC3数据集）
python scripts/build_real_dataset.py

# 训练模型并生成评估报告
python scripts/run_pipeline.py
```

训练完成后，模型文件保存在 `artifacts/models/detector_pipeline.joblib`。

### 2.3 启动服务

```bash
python -m src.system.app
```

启动成功后，在浏览器中打开 `http://127.0.0.1:5000` 即可进入系统。

### 3.1 文本输入区

页面中央的文本框用于输入待检测文本。支持直接输入、粘贴文本内容。

- **右侧计数器：** 实时显示当前已输入字符数 / 最大字符数

### 3.2 检测按钮

点击 **"开始检测"** 按钮后，系统开始分析文本。按钮在检测过程中会变为 **"检测中..."** 并暂时不可点击，检测完成后自动恢复。

### 3.3 结果展示区

检测完成后自动显示，包含以下信息：

| 项目 | 说明 |
|------|------|
| 标签 | "疑似人类书写" 或 "疑似机器生成" |
| 机器文本概率 | 模型判断该文本由AI生成的概率（0%–100%） |
| 置信度 | 模型对本次判断的把握程度（越接近100%越确定） |
| 贡献最大的特征 | 影响本次判断的前8个最重要的特征及其贡献度 |

## 4. 使用方法

### 4.1 基本检测流程

1. 在文本输入框中输入或粘贴待检测的中文文本
2. 确认字数满足要求（6–10000字符，右下角会显示实时字数）
3. 点击 **"开始检测"** 按钮
4. 等待检测完成，查看结果

### 4.2 输入要求

| 项目 | 要求 |
|------|------|
| 文本长度 | 6–10000 个字符 |
| 内容要求 | 至少包含 1 个中文/英文/数字词汇 |
| 特殊字符 | 特殊符号占比不超过 85% |
| 编码要求 | 文本中不能包含控制字符或空字符 |

### 4.3 结果解读

**如何理解检测结果：**

- **机器文本概率 > 50%：** 系统判定为"疑似机器生成"，概率越高表示AI特征越明显
- **机器文本概率 < 50%：** 系统判定为"疑似人类书写"，概率越低表示人类特征越明显
- **置信度 > 80%：** 系统对判断结果较为确定
- **置信度 < 70%：** 系统对判断结果不太确定，该文本可能处于人机风格的模糊地带

**如何理解特征贡献：**

每个特征条表示该特征对本次判断的贡献程度，条越长表示该特征的影响越大。

常见特征含义：

| 特征名 | 含义 | 人类文本通常 | 机器文本通常 |
|--------|------|------------|------------|
| 句首词多样性 | 句首用词的变化程度 | 较高（句首多变） | 较低（句首固定） |
| 句长标准差 | 句子长短的变化幅度 | 较大（长短句交替） | 较小（句式统一） |
| 字符多样性 | 用字的丰富程度 | 较高 | 较低 |
| 重复词元占比 | 重复使用相同词汇的比例 | 较低 | 较高 |
| 词频熵 | 词汇使用的不可预测程度 | 较高 | 较低 |

## 5. 常见问题

### 5.1 页面提示"模型尚未训练"

**原因：** 模型文件不存在，系统无法执行检测。

**解决方法：** 先运行训练脚本生成模型：

```bash
python scripts/run_pipeline.py
```

训练完成后刷新页面即可。

### 5.2 提示"文本过短，至少需要6个字符"

**原因：** 输入的文本长度不足6个字符。

**解决方法：** 输入更长的文本内容。注意空格和标点不计入有效字符。

### 5.3 提示"有效内容不足，至少需包含1个中文/英文/数字词"

**原因：** 输入的文本仅包含标点符号或特殊字符，没有实质内容。

**解决方法：** 输入包含中文、英文或数字的实际文本内容。

### 5.4 提示"特殊字符占比过高，请输入正常文本"

**原因：** 文本中特殊符号（如@#$%等）占比超过85%。

**解决方法：** 减少特殊符号的使用，输入正常的中文或英文文本。

### 5.5 检测结果不太准确

**可能原因：**
- 文本过短，特征信息不足
- 文本风格处于人机模糊地带（如非常正式的人类文本）
- 模型训练数据量有限

**建议：** 输入较长的文本（100字以上）可获得更可靠的检测结果。
